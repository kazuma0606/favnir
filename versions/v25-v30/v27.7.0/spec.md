# v27.7.0 仕様書 — `fav infer --from delta` / `--from iceberg`

## 概要

`fav infer` コマンドに `--from delta` / `--from iceberg` サブコマンドを追加する。
Delta Lake / Apache Iceberg テーブルのスキーマから Favnir の型定義（`type Row = { ... }`）を自動生成する。

---

## 背景

ロードマップ v27.7「`fav infer --from delta` / `--from iceberg`」より。Data Lakehouse フェーズの第 7 コンポーネント。
「既存の Delta テーブルに繋ぎたい」という最初の 5 分の体験を改善する。

v27.7.0 は stub 実装。`DeltaLake.infer_schema_raw` / `Iceberg.infer_schema_raw` は引数検証のみを行い、
固定のスキーマ JSON を返す。実際のテーブルスキーマ読み取りは v28.x（`delta-rs` / `iceberg-rust` 統合時）に延期する。

---

## CLI 変更

### 新フラグ（main.rs `infer` アームに追加）

| フラグ | 説明 |
|---|---|
| `--path <path>` | Delta テーブルのパス（例: `/tmp/orders` / `s3://bucket/orders`） |
| `--catalog <url>` | Iceberg REST カタログ URL（例: `http://localhost:8181`） |

### 使用例

```bash
# Delta Lake テーブルからスキーマを自動生成
fav infer --from delta --path /tmp/orders

# Iceberg テーブルから（REST カタログ）
fav infer --from iceberg --catalog http://localhost:8181 --table warehouse.orders
```

### 出力例

```favnir
// Generated by fav infer --from delta --path /tmp/orders
type OrdersRow = {
  id: Int,
  name: String,
  amount: Float,
  created_at: DateTime,
  active: Bool,
}
```

## 型マッピング

| Delta / Iceberg 型 | Favnir 型 |
|---|---|
| `long` / `integer` / `int` | `Int` |
| `double` / `float` | `Float` |
| `string` | `String` |
| `timestamp` / `timestamp_ntz` | `DateTime` |
| `boolean` | `Bool` |
| その他 / 不明 | `String`（デフォルト） |

### 3. `fav/src/main.rs` — `infer` アームに `--path` / `--catalog` フラグと dispatch 追加

`--from delta` / `--from iceberg` の既存 snowflake/postgres パターンに倣った dispatch を追加。
`--table` フラグは既存実装済み（`table_name` 変数に格納）のため追加不要。
`use driver::{..., cmd_infer_delta, cmd_infer_iceberg}` のインポートも追加すること。

```rust
// --path フラグ（delta 用）
"--path" => { path_arg = Some(args[i+1].clone()); i += 2; }
// --catalog フラグ（iceberg 用）
"--catalog" => { catalog_arg = Some(args[i+1].clone()); i += 2; }

// dispatch
if from_source.as_deref() == Some("delta") {
    let path = path_arg.as_deref().unwrap_or_else(|| { eprintln!(...); exit(1) });
    cmd_infer_delta(path, out_path.as_deref());
    return;
}
if from_source.as_deref() == Some("iceberg") {
    let catalog = catalog_arg.as_deref().unwrap_or_else(|| { eprintln!(...); exit(1) });
    let table = table_name.as_deref().unwrap_or_else(|| { eprintln!(...); exit(1) });
    cmd_infer_iceberg(catalog, table, out_path.as_deref());
    return;
}
```

  active: Bool,

stub のため固定フィールド 5 件を出力する。v28.x で `delta-rs` からの実スキーマ読み取りに置換する。

## スコープ外（v28.x 以降）

- `delta-rs` クレートを使った実 Delta Lake スキーマ読み取り
- `iceberg-rust` クレートを使った実 Iceberg スキーマ読み取り
- `fav infer --from delta --path /tmp/test_delta_table` での実コード出力検証（実スキーマ依存）
- Nested 型（`struct` / `array` / `map`）の Favnir 型マッピング
- `null` / `nullable` フィールドの `Option<T>` マッピング
- `date` / `binary` / `decimal` 型の専用マッピング（v27.7.0 では `String` にフォールバック）
