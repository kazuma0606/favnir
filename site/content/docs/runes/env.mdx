# Env Rune

エフェクト: `!Env`

```favnir
import rune "env"
```

## 環境変数の取得

```favnir
public fn main() -> Unit !Io !Env {
  // 文字列として取得（Option<String>）
  let api_key = env.get("API_KEY")

  // 既定値付き
  let region = env.get_or("AWS_REGION", "ap-northeast-1")

  // 必須（なければエラー）
  bind db_url <- env.require("DATABASE_URL")

  // Int に変換
  let port = env.get_int_or("PORT", 8080)
}
```

## .env ファイル

```favnir
env.load_dotenv(".env.local")
```

```bash
# .env.local
DATABASE_URL=sqlite://./dev.db
API_KEY=dev-key-12345
PORT=3000
```

## fav.toml 設定

```toml
[env]
dotenv = ".env"
prefix = "APP_"   # APP_ プレフィックスを持つ変数のみ対象
```
