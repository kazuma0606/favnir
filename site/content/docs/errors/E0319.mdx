# E0319: missing !Stream effect

## 概要

Kafka / MSK ストリーミング操作を行う関数が `!Stream` エフェクトを宣言していないときに発生します。

## エラー例

```fav
import "runes/kafka/kafka"

public fn publish(msg: String) -> Result<Unit, String> {  // E0319: missing !Stream effect
  chain _ <- Kafka.produce("my-topic", msg)
  ok(unit)
}
```

## 修正方法

`!Stream` エフェクトを関数シグネチャに追加する:

```fav
import "runes/kafka/kafka"

public fn publish(msg: String) !Stream -> Result<Unit, String> {  // OK
  chain _ <- Kafka.produce("my-topic", msg)
  ok(unit)
}
```
