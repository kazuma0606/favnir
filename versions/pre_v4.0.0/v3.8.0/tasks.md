# Favnir v3.8.0 Tasks

## Phase 0: バージョン更新 + 依存追加

- [x] `fav/Cargo.toml` の version を `"3.8.0"` に更新
- [x] `fav/src/main.rs` のヘルプテキスト・バージョン文字列を更新
