# ClickHouse バルクインサート

ClickHouse への大量データの高速バルクインサートパイプラインです。

```favnir
import runes/clickhouse

stage BulkInsertClickHouse: List<Event> -> Unit !Db = |events| {
  bind conn <- ClickHouse.connect(config.clickhouse_url)
  ClickHouse.insert_batch(conn, config.clickhouse_table, events)
}
```
