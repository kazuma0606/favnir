# Vertex AI オンライン推論

Google Vertex AI のデプロイ済みモデルエンドポイントを Favnir パイプラインから呼び出します。

```favnir
import runes/vertex-ai

stage ScoreWithVertexAI: List<Feature> -> List<Prediction> = |features| {
  bind predictions <- VertexAI.predict(
    config.vertex_endpoint,
    features |> List.map(encode_feature)
  )
  Result.ok(predictions)
}
```
