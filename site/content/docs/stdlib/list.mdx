## StdList 高レベル関数（runes/stdlib/list.fav）

`import StdList from "stdlib/list"` でインポートして使用します。

| 関数 | シグネチャ | 説明 |
|------|-----------|------|
| `StdList.head` | `List<A> -> Option<A>` | 先頭要素（`List.first` のエイリアス） |
| `StdList.tail` | `List<A> -> List<A>` | 先頭を除いたリスト |
| `StdList.group_by` | `List<A>, A -> String -> Map<String, List<A>>` | キー関数でグループ化 |
| `StdList.zip_with` | `List<A>, List<B>, A -> C -> List<C>` | 2 リストを要素ごとに合成 |
| `StdList.sort_by` | `List<A>, A -> String -> List<A>` | String キーでソート |
| `StdList.intersperse` | `List<A>, A -> List<A>` | 要素間にセパレータを挿入 |

```favnir
bind users <- DB.query_raw("SELECT name, dept FROM users")

// 部署ごとにグループ化
bind by_dept <- StdList.group_by(users, |u| u.dept)

// 名前順にソート
bind sorted <- StdList.sort_by(users, |u| u.name)

// カンマ区切りの名前リスト
bind names  <- List.map(users, |u| u.name)
bind csv    <- StdList.intersperse(names, ",")
```

## 使用例

```favnir
// nums は外部データ（DuckDB クエリ結果など）から取得した List<Int>
fn demo(nums: List<Int>) -> Unit {
  bind doubled    <- List.map(nums, |n| n * 2)
  // 各要素が2倍のリスト

  bind evens      <- List.filter(nums, |n| n % 2 == 0)
  // 偶数のみのリスト

  bind sum        <- List.fold(nums, 0, |acc, n| acc + n)
  // 合計値

  bind has_three  <- List.any(nums, |n| n == 3)
  // 3 が含まれるか

  bind first_even <- List.find(nums, |n| n % 2 == 0)
  // Option.some(最初の偶数) または Option.none
  IO.println_int(sum)
}
```

## v16.4.0 追加関数

### グループ化・集計

```fav
-- グループ化（fn は String キーを返す）
bind groups <- List.group_by(|r| r.category, rows)   -- Map<String, List<Row>>

-- 条件付き件数
bind count <- List.count_where(rows, |r| r.active)   -- Int

-- Float 合計
bind total <- List.sum_by(rows, |r| r.amount)         -- Float

-- 最大値・最小値要素
bind best  <- List.max_by(rows, |r| r.score)          -- Option<Row>
bind worst <- List.min_by(rows, |r| r.score)          -- Option<Row>
```

### ソート

```fav
bind sorted      <- List.sort_by(rows, |r| r.amount)      -- 昇順
bind sorted_desc <- List.sort_by_desc(rows, |r| r.amount) -- 降順
```

### 変換・フィルタ

```fav
-- 重複除去
bind unique    <- List.distinct(xs)                    -- List<T>（== による）
bind unique_by <- List.distinct_by(xs, |r| r.id)      -- キー指定

-- バッチ分割
bind batches <- List.chunk(rows, 100)                  -- List<List<T>>

-- zip 拡張
bind zipped   <- List.zip_with(as, bs, |a b| merge(a, b))  -- List<C>
bind unzipped <- List.unzip(pairs)                          -- Pair<List<A>, List<B>>
```
