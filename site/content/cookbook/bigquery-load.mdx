# BigQuery データロード

変換済みデータを BigQuery テーブルにロードするパイプラインです。

```favnir
import runes/bigquery

stage LoadToBigQuery: List<Row> -> Unit = |rows| {
  bind client <- BigQuery.client(config.bigquery_project, config.bigquery_dataset)
  BigQuery.insert_rows(client, config.bigquery_table, rows)
}
```
