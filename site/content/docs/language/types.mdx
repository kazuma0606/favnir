# 型システム

## プリミティブ型

| 型 | 説明 | 例 |
|----|------|----|
| `Int` | 64bit 整数 | `42`, `-1` |
| `Float` | 64bit 浮動小数点 | `3.14`, `-0.5` |
| `String` | UTF-8 文字列 | `"hello"` |
| `Bool` | 真偽値 | `true`, `false` |
| `Unit` | 値なし（副作用のある関数の戻り値） | `()` |

## Record 型

```favnir
type Point = { x: Int  y: Int }
type User  = { name: String  age: Int  email: Option<String> }

bind p <- Point { x: 3  y: 4 }
IO.println(p.x)   // 3
```

## List 型

```favnir
// List は外部データ（クエリ結果など）や List.map 等で構築します
fn demo(nums: List<Int>) -> Unit !Io {
  bind doubled <- List.map(nums, |n| n * 2)   // 各要素を2倍
  bind evens   <- List.filter(nums, |n| n % 2 == 0)  // 偶数のみ
  bind sum     <- List.fold(nums, 0, |acc, n| acc + n)  // 合計
  IO.println_int(sum)
}
```

## Option 型

値が存在しないかもしれないことを型で表現します:

```favnir
fn find_user(id: Int) -> Option<User> {
  if id == 1 { Option.some(User { name: "田中"  age: 30  email: Option.none }) } else { Option.none }
}

match find_user(1) {
  Some(u) => IO.println(u.name)
  None    => IO.println("ユーザーが見つかりません")
}

// Option.unwrap_or で既定値
bind name <- Option.unwrap_or(find_user(1), User { name: "Unknown"  age: 0  email: Option.none })
```

## Result 型

成功か失敗かを型で表現します:

```favnir
fn parse_int(s: String) -> Result<Int, String> {
  // ...
}

match parse_int("42") {
  Ok(n)  => IO.println($"数値: {n}")
  Err(e) => IO.println($"エラー: {e}")
}

// and_then でチェーン
bind result <- Result.and_then(parse_int("42"), |n| Result.ok(n + 1))
IO.println(Result.unwrap_or(result, 0))
```

## ジェネリック型

```favnir
fn first<T>(list: List<T>) -> Option<T> {
  if List.length(list) == 0 { Option.none } else { List.first(list) }
}
```
