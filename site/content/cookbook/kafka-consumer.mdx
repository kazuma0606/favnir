# Kafka コンシューマーパイプライン

Kafka トピックからメッセージを消費し、変換してデータウェアハウスに書き込むストリーミングパイプラインです。

```favnir
import runes/kafka

seq KafkaToWarehouse =
  ConsumeKafka
  |> TransformEvent
  |> WriteToWarehouse

stage ConsumeKafka: Unit -> List<KafkaMessage> = |_| {
  bind consumer <- Kafka.consumer(config.kafka_brokers, config.kafka_group_id)
  Kafka.poll(consumer, config.kafka_topic, 100)
}
```
