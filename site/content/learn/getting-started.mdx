# はじめての Favnir

## インストール

```bash
cargo install fav
fav --version  # favnir 24.7.0
```

## Hello World

`hello.fav` を作成します:

```favnir
fn main() -> String { "Hello, Favnir!" }
```

実行:

```bash
fav run hello.fav
# Hello, Favnir!
```

## 最初のパイプライン

```favnir
stage Load   -> String { "data.csv" }
stage Process(s: String) -> String { s }

pipeline Main {
  Load |> Process
}
```

```bash
fav run pipeline.fav
```

## Capability Context を使う

v34.5.0 以降、副作用は `AppCtx` パラメータで表現します:

```favnir
import runes/ctx

fn fetch_and_save(ctx: AppCtx, url: String) -> Unit {
    bind { http, io } <- ctx
    bind data <- http.get(url)
    io.println(data)
}
```

詳細は [ctx 構文リファレンス](/docs/ctx-syntax-guide) を参照。
