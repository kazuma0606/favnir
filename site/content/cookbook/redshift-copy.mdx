# Redshift COPY コマンド連携

S3 に配置したファイルを Redshift COPY コマンドで高速ロードします。

```favnir
import runes/redshift

stage CopyToRedshift: String -> Unit = |s3_path| {
  bind conn <- Redshift.connect(config.redshift_conn)
  Redshift.copy(conn, config.redshift_table, s3_path, config.iam_role)
}
```
