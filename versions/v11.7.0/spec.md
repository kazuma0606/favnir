# Favnir v11.7.0 仕様書

作成日: 2026-06-06
テーマ: uv 統合 — `fav transpile` が uv プロジェクトとして完結する出力を生成

---

## 背景と目的

v11.6.0 で `Postgres.*` → psycopg2 変換が完了した。
v11.7.0 では `fav transpile` コマンドを拡張し、
生成 Python コードを `uv run` で即実行できる完結したプロジェクト構造として出力する。

新しい 3 つのオプションを追加する:
- `--out-dir <dir>`: ディレクトリに `main.py` + `pyproject.toml` + `README.md` を生成
- `--check`: 生成後に `python -m py_compile` で構文検証（exit 1 on fail）
- `--run`: 生成後に `uv run main.py` を実行

---

## 新規 CLI オプション

### `--out-dir <dir>`

```
fav transpile --target python <file.fav> --out-dir <dir>
```

`<dir>/` 以下に以下を生成する:
- `main.py` — トランスパイル済み Python コード
- `pyproject.toml` — uv 形式（依存自動追加）
- `README.md` — 生成元 .fav ファイルの情報

`--out-dir` がない場合は従来通り `<file>.py` をカレントか `--out` 指定先に生成。

### `--check`

```
fav transpile --target python <file.fav> --check
```

生成後に Python 構文検証を実行する:

```rust
std::process::Command::new("python")
    .args(["-m", "py_compile", &out_file])
```

構文エラーがあれば `exit 1`、なければ `fav transpile: syntax ok` を出力。

### `--run`

```
fav transpile --target python <file.fav> --out-dir out --run
```

生成後に `uv run main.py` を実行する:

```rust
std::process::Command::new("uv")
    .args(["run", "main.py"])
    .current_dir(&out_dir)
```

`uv` が見つからない場合は `error: uv not found (install: pip install uv)` を表示して exit 1。

## 生成ファイル: pyproject.toml（uv 形式）

`--out-dir` モードでは uv 形式の pyproject.toml を生成する:

```toml
[project]
name = "<basename>"
version = "0.1.0"
requires-python = ">=3.11"
dependencies = [
    "boto3>=1.34",          # import boto3 がある場合
    "psycopg2-binary>=2.9", # import psycopg2 がある場合
]

[build-system]
requires = ["hatchling"]
build-backend = "hatchling.build"
```

`<basename>` は `.fav` ファイルのベース名（例: `pipeline.fav` → `pipeline`）。

## 生成ファイル: README.md

`--out-dir` モードで自動生成:

```markdown
# <basename>

Generated by `fav transpile --target python`.

## Source
- Original: `<input_path>`
- Generated: `main.py`

## Requirements
- Python >= 3.11
- [uv](https://github.com/astral-sh/uv)

## Run
```
uv run main.py
```
```

## テスト設計（v11700_tests）

| テスト名 | 検証内容 |
|---|---|
| `transpile_out_dir_creates_main_py` | `--out-dir` で `main.py` が生成される |
| `transpile_out_dir_creates_pyproject` | `--out-dir` で `pyproject.toml` が生成される |
| `transpile_out_dir_creates_readme` | `--out-dir` で `README.md` が生成される |
| `transpile_check_valid_python` | `--check` で構文正常時に exit 0 |
| `transpile_pyproject_project_name` | `pyproject.toml` の name が fav ファイル名から生成される |
| `transpile_pyproject_has_correct_deps` | boto3 / psycopg2 が依存に含まれる |
