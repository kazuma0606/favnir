# AWS Secrets Manager から認証情報取得

AWS Secrets Manager に格納した認証情報をランタイムに取得し、Rune の接続設定に注入します。

## コード例

```favnir
import runes/aws
import runes/postgres

type DbSecret = { username: String, password: String, host: String, port: Int, dbname: String }

fn get_db_url(secret_name: String) -> Result<String, String> {
    bind secret <- AWS.get_secret_json(secret_name, DbSecret)
    let url = String.concat([
        "postgres://", secret.username, ":", secret.password,
        "@", secret.host, ":", Int.to_string(secret.port), "/", secret.dbname
    ])
    Result.ok(url)
}

fn main() -> Unit {
    bind db_url <- get_db_url("prod/myapp/postgres")
    bind _      <- Postgres.connect(db_url)
    IO.println("Connected to database via Secrets Manager.")
}
```

## ポイント

- `AWS.get_secret_json` は Secrets Manager のシークレットを JSON として取得し、指定型にデシリアライズします。
- IAM ロールで `secretsmanager:GetSecretValue` 権限を付与してください。
- ローカル開発時は `AWS_PROFILE` または `AWS_ACCESS_KEY_ID` / `AWS_SECRET_ACCESS_KEY` 環境変数を設定します。
