# AOT コンパイル（ネイティブバイナリ）

`fav build --target native` を使うと、Favnir プログラムを直接実行可能なネイティブバイナリとして出力できます。

## VM モードとの比較

| モード | コマンド | 特徴 |
|---|---|---|
| VM（インタープリタ） | `fav run main.fav` | 起動が速い、デバッグ向き |
| バイトコード | `fav build --target vm` | 事前コンパイル、VM で実行 |
| **ネイティブ（AOT）** | `fav build --target native` | 最速実行、`fav` コマンド不要 |

## 基本的な使い方

```bash
# ネイティブバイナリとしてビルド
fav build --target native src/pipeline.fav -o pipeline

# 直接実行（fav run 不要）
./pipeline
```

## 前提条件

`cc`（GCC または Clang）がインストールされていること:

```bash
# macOS
xcode-select --install

# Ubuntu / Debian
sudo apt install gcc

# Fedora / RHEL
sudo dnf install gcc
```

## 対応している機能（v19.2.0）

| 機能 | 対応 |
|---|---|
| `Int` / `Bool` リテラル | ✓ |
| 算術演算（`+` `-` `*` `/`） | ✓ |
| 比較演算（`==` `!=` `<` `>` `<=` `>=`） | ✓ |
| 論理演算（`&&` `\|\|`） | ✓ |
| `if / else` 式 | ✓ |
| ローカル変数（`bind`） | ✓ |
| `List` / `String` / `Closure` | v19.3.0 以降 |
| エフェクト（`!IO` 等） | v19.4.0 以降 |

## 制限事項（v19.2.0）

- 対応する関数は `fn main() -> Int { ... }` 形式のみ（他の関数は v19.3.0 以降）
- `List`, `String`, `Map`, `Closure` 型は未対応
- クロスコンパイル（`--target x86_64-unknown-linux-musl` 等）は v19.3.0 以降

## 内部実装

```
.fav → Parser → AST → Compiler → IRProgram → Cranelift IR → .o → cc → 実行バイナリ
```

Cranelift（`cranelift-codegen 0.117`）を使用して Favnir IR を CLIF（Cranelift IR）に変換し、
ネイティブオブジェクトファイルを生成します。最後に `cc` でリンクして実行バイナリを生成します。
