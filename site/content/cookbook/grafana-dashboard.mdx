# Grafana ダッシュボード連携

Favnir パイプラインのメトリクスを Grafana ダッシュボードに送信します。

```favnir
import runes/grafana

stage PushToGrafana: PipelineStats -> Unit = |stats| {
  Grafana.push_metrics(
    config.grafana_url,
    config.grafana_api_key,
    [
      { "name": "rows_processed", "value": Float.from_int(stats.rows), "tags": ["pipeline:etl"] },
      { "name": "error_rate",     "value": stats.error_rate,           "tags": ["pipeline:etl"] }
    ]
  )
}
```
