# Math

Math モジュールは数値演算のためのユーティリティ関数を提供します。

## 基本演算

```fav
Math.abs(n)          -- Int → Int（絶対値）
Math.abs_float(f)    -- Float → Float

Math.round(f)        -- Float → Int（四捨五入）
Math.ceil(f)         -- Float → Int（切り上げ）
Math.floor(f)        -- Float → Int（切り捨て）
Math.round_to(f, 2)  -- Float → Float（小数点以下 n 桁）(v16.4.0)
```

## 比較・範囲

```fav
Math.min(a, b)           -- Int
Math.max(a, b)           -- Int
Math.min_float(a, b)     -- Float
Math.max_float(a, b)     -- Float
Math.clamp(v, lo, hi)    -- Int（lo <= v <= hi に収める）
```

## べき乗・対数

```fav
Math.pow(base, exp)      -- Int（整数べき乗、exp >= 0）
Math.pow_float(b, e)     -- Float
Math.sqrt(f)             -- Float

Math.log(f)              -- Float（自然対数）(v16.4.0)
Math.log2(f)             -- Float (v16.4.0)
Math.log10(f)            -- Float (v16.4.0)
```

## 定数

```fav
Math.pi                  -- Float（円周率）
Math.e                   -- Float（ネイピア数）
```

## 使用例

```fav
-- 小数点 2 桁に丸める
bind price  <- Math.round_to(99.9999, 2)   -- 100.0

-- 自然対数
bind ln2 <- Math.log(2.0)   -- 0.693...

-- clamp でスコアを 0〜100 に収める
bind clamped <- Math.clamp(raw_score, 0, 100)
```
