# Post-v1 Notes

This directory holds Favnir notes that are intentionally outside the v1.0.0 completion scope.

## Includes

- future language ergonomics
- validation and `stat` rune families
- graph/export ideas
- self-hosting expansion beyond the v1 entry point
- Forge comparison notes that still inform future design
- roadmap and gap-analysis material for post-v1 work

## Rule of Thumb

If a note is about making Favnir more expressive, more powerful, or more product-ready after the core v1 language/runtime/tooling is finished, it belongs here.
