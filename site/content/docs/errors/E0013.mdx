# E0013: where constraint failed

## 概要

`where` 句で定義されたバリデーション制約に違反する値を渡したときに発生します。

## エラー例

```fav
type Age(Int) where self > 0 and self < 150

public fn main() -> Age {
  Age(-5)  // E0013: where constraint failed
}
```

## 修正方法

```fav
type Age(Int) where self > 0 and self < 150

public fn main() -> Age {
  Age(25)  // OK
}
```
