# Playground v2

Favnir の [Playground](https://play.favnir.dev/playground) では、ブラウザ内で Favnir コードを書いて型チェック・実行できます。
v21.6.0 で「共有・テンプレート・ライブ統計」機能を追加しました。

## 基本機能

| ボタン | 動作 |
|---|---|
| **型チェック** | 型エラー・lint 警告を診断パネルに表示 |
| **▶ 実行** | WASM コンパイル → 実行 → 出力を表示 |
| **テンプレート ▼** | サンプルコードをエディタに挿入 |
| **📋 共有** | 現在のコードの共有 URL をクリップボードにコピー |

---

## 共有 URL（v21.6.0 新機能）

「📋 共有」ボタンを押すと、現在のコードがエンコードされた URL がクリップボードにコピーされます。

```
https://play.favnir.dev/playground?c=H4sIAAAAA...
```

- **サーバーレス方式**: コードを gzip 圧縮 + base64url エンコードして URL パラメータに埋め込みます。サーバー不要で即座に共有できます。
- **短縮 URL（Lambda）**: API が設定されている場合、`?s=abc123` 形式の短い URL が生成されます。

### フォーク（共有 URL からの読み込み）

共有 URL を開くと、エディタにコードが読み込まれ、以下のバナーが表示されます：

> 「共有コードを読み込みました — 自由に編集できます」

読み込んだコードはすぐに編集できます。変更を共有したいときは再度「📋 共有」を押してください。

## テンプレートギャラリー（v21.6.0 新機能）

「テンプレート ▼」ボタンから 6 種類のサンプルコードを選べます。

| テンプレート | 内容 |
|---|---|
| Hello World | `IO.println` で文字列を出力 |
| パイプライン基礎 | `stage` + `seq` + `\|>` の組み合わせ |
| List 操作 | `List.map` / `filter` / `fold` の活用 |
| Result 型 | `Result<T, E>` + `?` 演算子によるエラーハンドリング |
| レコード型 | `type` 定義 + フィールドアクセス |
| f-string フォーマット | f-string による文字列補間 |

---

## 実行統計（v21.6.0 新機能）

「▶ 実行」後、出力パネルの下部に実行時間が表示されます：

```
実行時間: 12ms  |  メモリ: 1.2MB
```

- **実行時間**: WASM コンパイル → instantiate → `main()` 実行までの合計時間
- **メモリ**: JS ヒープ使用量（Chrome のみ表示）

## 対応コード

Playground は WASM（WebAssembly）でブラウザ内実行するため、以下の制限があります：

**対応:**
- `Int` / `Float` / `Bool` / `String` / `Unit` 型
- `stage` / `seq` / `|>` パイプライン
- `IO.println` / `IO.println_int` / `IO.println_float` / `IO.println_bool`
- `List<Int>` 等の基本的なリスト操作

**ブラウザ実行非対応（`fav run` は対応）:**
- `match` 式・`Option` / `Result` 型
- ユーザー定義型（`type` 宣言）
- Rune（`use http`, `use postgres` 等）
- ファイル I/O

---

## ローカルでの実行

本格的な開発には `fav run` を使ってください：

```bash
fav run src/main.fav
```

詳細は [CLI リファレンス](/docs/cli/run) を参照してください。
