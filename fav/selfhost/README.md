# Favnir セルフホスト移植

Favnir コンパイラを Favnir 自身で書き直すプロジェクト。
Rust 製の VM 上で動く Favnir 製コンパイラを段階的に構築する。

## 現在の状態

| コンポーネント | 状態 | 場所 |
|---|---|---|
| レキサー（算術演算子のみ） | ✅ 動作 | `lexer/` |
| レキサー（完全版） | ❌ 未着手 | — |
| パーサー | ❌ 未着手 | — |
| 型検査器 | ❌ 未着手 | — |
| コンパイラ | ❌ 未着手 | — |

## 実行方法

```sh
# レキサーを実行
fav run selfhost/lexer/lexer.fav

# レキサーのテストを実行
fav test selfhost/lexer/lexer.test.fav
```

## 移植方針

- コンポーネントごとにサブディレクトリを切る（`lexer/`, `parser/`, `checker/`, ...）
- 各コンポーネントに `*.fav` 実装と `*.test.fav` テストを置く
- Rust 製の同等コンポーネントと並行して動作確認しながら進める
