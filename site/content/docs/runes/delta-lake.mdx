# Delta Lake Rune

`import rune "delta-lake"` で Delta Lake テーブルの読み書きができます。

> **注**: v27.1.0 は引数検証 stub 実装です。`delta-rs` クレートを使った実テーブル連携は v28.x で実装予定です。

## セットアップ

```toml
# fav.toml
[runes]
delta-lake = "27.1.0"
```

```favnir
import rune "delta-lake"
```

## エフェクト

Delta Lake はローカルファイルシステムおよび S3 のファイル操作を行うため `!Io` エフェクトを使用します。

## 関数リファレンス

### `DeltaLake.read`

```favnir
fn read(path: String) -> Result<String, String> !Io
```

Delta テーブル全件を JSON 配列文字列として読み込みます。

```favnir
bind rows <- DeltaLake.read("/tmp/favnir_delta/orders")
```

### `DeltaLake.read_with_filter`

```favnir
fn read_with_filter(path: String, predicate: String) -> Result<String, String> !Io
```

述語プッシュダウン付きで読み込みます（v27.1.0 は stub）。

```favnir
bind rows <- DeltaLake.read_with_filter("/tmp/favnir_delta/orders", "amount > 100")
```

### `DeltaLake.write`

```favnir
fn write(path: String, data: String, mode: String) -> Result<Unit, String> !Io
```

Delta テーブルに書き込みます。`mode` は `"append"` または `"overwrite"` のみ有効です。

```favnir
bind _ <- DeltaLake.write("/tmp/favnir_delta/orders", rows_json, "append")
```

### `DeltaLake.merge`

```favnir
fn merge(path: String, data: String, condition: String) -> Result<String, String> !Io
```

MERGE（upsert / delete-when-matched）を実行します。結果は JSON 文字列で返ります。

```favnir
bind result <- DeltaLake.merge("/tmp/favnir_delta/orders", new_rows, "source.id = target.id")
```

### `DeltaLake.history`

```favnir
fn history(path: String) -> Result<String, String> !Io
```

Delta テーブルのトランザクションログ（バージョン一覧）を JSON 配列として取得します。

```favnir
bind log <- DeltaLake.history("/tmp/favnir_delta/orders")
```

### `DeltaLake.vacuum`

```favnir
fn vacuum(path: String, retention_hours: Int) -> Result<Unit, String> !Io
```

指定した保持時間より古いファイルを削除します。Delta Lake 仕様により `retention_hours` は最小 **168（7 日）** 以上が必要です。

```favnir
bind _ <- DeltaLake.vacuum("/tmp/favnir_delta/orders", 336)
```

### `DeltaLake.optimize`

```favnir
fn optimize(path: String) -> Result<String, String> !Io
```

小さいファイルをまとめるコンパクションを実行します。結果は JSON 文字列で返ります。

```favnir
bind result <- DeltaLake.optimize("/tmp/favnir_delta/orders")
```

## パイプライン例

```favnir
// examples/delta_lake_etl.fav
import rune "delta-lake"

stage LoadRawData: Unit -> Result<String, String> !Io = |_| {
    DeltaLake.read("/tmp/favnir_delta/raw_orders")
}

stage TransformOrders: String -> Result<String, String> = |rows_json| {
    if String.length(rows_json) > 2
    then Result.ok(rows_json)
    else Result.err("empty delta table")
}

stage SaveProcessed: String -> Result<String, String> !Io = |data| {
    bind _ <- DeltaLake.write("/tmp/favnir_delta/processed_orders", data, "append")
    Result.ok("written to delta table")
}

seq DeltaEtlPipeline = LoadRawData |> TransformOrders |> SaveProcessed
```

## スコープ外（v28.x 以降）

- `delta-rs` クレートを使った実 Delta テーブルの読み書き
- `DeltaLake.read[T]` ジェネリック API
- S3 / LocalStack 上の Delta テーブル操作
- Time travel（バージョン指定読み込み）
- Change Data Feed（CDC）
