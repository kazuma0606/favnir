# SageMaker エンドポイント推論

AWS SageMaker のデプロイ済みエンドポイントを Favnir パイプラインから呼び出して推論を実行します。

```favnir
import runes/sagemaker

stage InferWithSageMaker: String -> String = |payload| {
  SageMaker.invoke(config.sagemaker_endpoint, payload)
}

stage BatchScoreWithSageMaker: List<String> -> List<String> = |payloads| {
  payloads |> List.map(|p| SageMaker.invoke(config.sagemaker_endpoint, p)) |> Result.all
}
```
