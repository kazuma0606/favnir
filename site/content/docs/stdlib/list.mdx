# List

## 関数一覧

| 関数 | シグネチャ | 説明 |
|------|-----------|------|
| `List.length` | `List<T> -> Int` | 要素数 |
| `List.first` | `List<T> -> Option<T>` | 先頭要素（なければ None） |
| `List.map` | `List<T>, T -> U -> List<U>` | 変換 |
| `List.filter` | `List<T>, T -> Bool -> List<T>` | フィルタ |
| `List.fold` | `List<T>, U, (U, T) -> U -> U` | 集約 |
| `List.any` | `List<T>, T -> Bool -> Bool` | いずれかが条件を満たすか |
| `List.all` | `List<T>, T -> Bool -> Bool` | すべてが条件を満たすか |
| `List.find` | `List<T>, T -> Bool -> Option<T>` | 条件を満たす最初の要素 |
| `List.append` | `List<T>, T -> List<T>` | 末尾に追加 |
| `List.concat` | `List<T>, List<T> -> List<T>` | 結合 |
| `List.reverse` | `List<T> -> List<T>` | 逆順 |
| `List.take` | `List<T>, Int -> List<T>` | 先頭 N 件 |
| `List.drop` | `List<T>, Int -> List<T>` | 先頭 N 件を除く |
| `List.zip` | `List<T>, List<U> -> List<(T, U)>` | ペアにする |
| `List.flat_map` | `List<T>, T -> List<U> -> List<U>` | flatten + map |

## 使用例

```favnir
// nums は外部データ（DuckDB クエリ結果など）から取得した List<Int>
fn demo(nums: List<Int>) -> Unit !Io {
  bind doubled    <- List.map(nums, |n| n * 2)
  // 各要素が2倍のリスト

  bind evens      <- List.filter(nums, |n| n % 2 == 0)
  // 偶数のみのリスト

  bind sum        <- List.fold(nums, 0, |acc, n| acc + n)
  // 合計値

  bind has_three  <- List.any(nums, |n| n == 3)
  // 3 が含まれるか

  bind first_even <- List.find(nums, |n| n % 2 == 0)
  // Option.some(最初の偶数) または Option.none
  IO.println_int(sum)
}
```
