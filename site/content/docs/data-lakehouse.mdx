# Data Lakehouse

**宣言日**: 2026-06-27
**宣言バージョン**: v28.0.0

---

## 宣言

> 「Delta Lake テーブルを Favnir から型安全に読み書きし、
>  dbt モデルの結果を次のステージに渡す」
> = Data Lakehouse の完成を象徴するデモ

v28.0.0 をもって、Favnir の **Data Lakehouse** マイルストーンを正式に宣言します。

現代のデータ基盤の中心は「データレイクハウス」です。
Delta Lake / Apache Iceberg がデータレイクの標準になり、
dbt がデータ変換の標準ツールとして普及しています。
このマイルストーンで Favnir は現代データ基盤アーキテクチャに完全に統合されました。

## Data Lakehouse の定義

本プロジェクト固有の定義:

> 「Delta Lake / Iceberg テーブルの読み書きができ、dbt モデルを参照でき、
>  主要 DWH 3 本（BigQuery / Redshift / ClickHouse）に接続できる」状態

---

## 象徴デモ

```favnir
import rune "delta-lake"
import rune "dbt"
import rune "sqlite"

stage LoadFromDelta: Unit -> List<OrderRow> = |_| {
    DeltaLake.read[OrderRow]("s3://my-bucket/orders")
}

stage EnrichWithDbt: List<OrderRow> -> List<EnrichedOrder> = |orders| {
    bind summary <- Dbt.ref(config.dbt, "customer_summary")
    Result.ok(List.map(orders, |o| enrich(o, summary)))
}

stage SaveToSqlite: List<EnrichedOrder> -> Unit = |rows| {
    bind db <- SQLite.open_memory()
    bind _  <- SQLite.execute(db, "CREATE TABLE orders (id INT, amount REAL)", "[]")
    SQLite.execute_many(db, "INSERT INTO orders VALUES (?, ?)", rows)
}

seq DataLakehousePipeline = LoadFromDelta |> EnrichWithDbt |> SaveToSqlite
```
