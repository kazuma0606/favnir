# Language Maturity Overview

Favnir v50 は **Language Maturity** の宣言バージョンです。
v46 から v49 で積み重ねてきた機能が統合され、データ処理言語として成熟しました。

---

## 4 本柱（Four Pillars）

### 1. `return` ガード節

```favnir
fn process(order: Order) -> Result<Output, String> {
  return Result.err("invalid id") if order.id <= 0
  bind validated <- validate(order)
  Ok(transform(validated))
}
```

早期 return によるガード節パターンで、ネストを排除したフラットなコードが書けます。

### 2. 成熟した標準ライブラリ（Mature stdlib）

`List.*` / `Map.*` / `String.*` / `DateTime.*` が完備しています。
外部ライブラリなしに典型的なデータ変換処理を記述できます。

### 3. 明確なモジュールシステム（Clear Module System）

```favnir
import "./stages/validate" as validate
import kafka
```

ローカル import と Rune import が統一された構文で利用できます。

### 4. インラインテスト（Inline Testing）

```favnir
#[test]
fn test_process() {
  assert_ok(process(good_order))
  assert_err(process(bad_order))
}
```

本番コードと同じファイルにテストを記述でき、`fav test` で実行できます。
