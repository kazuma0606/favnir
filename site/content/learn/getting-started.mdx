# はじめての Favnir

## インストール

```bash
cargo install fav
fav --version  # favnir 24.7.0
```

## Hello World

`hello.fav` を作成します:

```favnir
fn main() -> String { "Hello, Favnir!" }
```

実行:

```bash
fav run hello.fav
# Hello, Favnir!
```

## 最初のパイプライン

```favnir
stage Load   -> String { "data.csv" }
stage Process(s: String) -> String { s }

pipeline Main {
  Load |> Process
}
```

```bash
fav run pipeline.fav
```
