# MongoDB ドキュメント ETL

MongoDB コレクションから注文ドキュメントを読み込み、正規化して別コレクションに書き込みます。

## コード例

```favnir
import runes/mongodb

type RawDoc = { _id: String, customer_name: String, total: Float, status: String }
type CleanDoc = { id: String, customer: String, total: Float, active: Bool }

fn normalize(d: RawDoc) -> CleanDoc {
    {
        id:       d._id
        customer: String.trim(d.customer_name)
        total:    Float.round(d.total, 2)
        active:   d.status == "active"
    }
}

fn main() -> Unit {
    bind raw   <- MongoDB.find("orders_raw", {})
    let cleaned = List.map(raw, normalize)
    bind n     <- MongoDB.insert_many("orders_clean", cleaned)
    IO.println(String.concat(["Inserted ", Int.to_string(n), " documents."]))
}
```

## ポイント

- `MongoDB.find` の第二引数はクエリフィルタです。`{}` で全件取得します。
- 環境変数 `MONGODB_URI` で接続文字列を設定してください。
- 大量ドキュメントは `MongoDB.find_cursor` でカーソル方式に切り替えると定常メモリで処理できます。
