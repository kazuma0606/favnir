# Delta Lake Upsert パイプライン

Delta Lake テーブルへの upsert（merge）操作をパイプラインとして実装します。

```favnir
import runes/delta-lake

stage UpsertOrders: List<Order> -> Unit = |orders| {
  bind table <- DeltaLake.open(config.delta_table_path)
  DeltaLake.merge(table, orders, "target.id = source.id")
}
```
