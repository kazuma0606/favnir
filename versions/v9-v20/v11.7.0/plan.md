## Phase B — `--out-dir` サポート

`cmd_transpile` の出力決定ロジックを更新:

```rust
// --out-dir が指定された場合
if let Some(ref dir) = out_dir {
    let dir_path = std::path::Path::new(dir);
    std::fs::create_dir_all(dir_path).unwrap_or_else(|e| {
        eprintln!("error creating dir {}: {}", dir, e);
        std::process::exit(1);
    });

    // main.py
    let main_py_path = dir_path.join("main.py");
    if let Err(e) = std::fs::write(&main_py_path, &py_src) {
        eprintln!("error writing main.py: {}", e);
        std::process::exit(1);
    }
    println!("transpiled: {} -> {}/main.py", input, dir);

    // pyproject.toml（既存の場合はスキップ）
    let pyproject_path = dir_path.join("pyproject.toml");
    if !pyproject_path.exists() {
        let content = build_pyproject_content(&py_src, &basename);
        let _ = std::fs::write(&pyproject_path, &content);
        println!("generated: {}/pyproject.toml", dir);
    }

    // README.md
    let readme_path = dir_path.join("README.md");
    if !readme_path.exists() {
        let content = build_readme_content(&input, &basename);
        let _ = std::fs::write(&readme_path, &content);
        println!("generated: {}/README.md", dir);
    }
} else {
    // 従来の --out / デフォルト出力
    // ...既存コード...
}
```

### `basename` の決定

```rust
let basename = std::path::Path::new(&input)
    .file_stem()
    .and_then(|s| s.to_str())
    .unwrap_or("transpiled")
    .to_string();
```

### `build_pyproject_content` ヘルパー

```rust
fn build_pyproject_content(py_src: &str, name: &str) -> String {
    let boto3_dep = if py_src.contains("import boto3") {
        "    \"boto3>=1.34\",\n"
    } else { "" };
    let psycopg2_dep = if py_src.contains("import psycopg2") {
        "    \"psycopg2-binary>=2.9\",\n"
    } else { "" };
    format!(
        "[project]\nname = \"{}\"\nversion = \"0.1.0\"\n\
         requires-python = \">=3.11\"\ndependencies = [\n{}{}]\n\n\
         [build-system]\nrequires = [\"hatchling\"]\n\
         build-backend = \"hatchling.build\"\n",
        name, boto3_dep, psycopg2_dep
    )
}
```

## Phase C — README.md 生成

fn build_readme_content(input_path: &str, name: &str) -> String {
    format!(
        "# {name}\n\n\
         Generated by `fav transpile --target python`.\n\n\
         ## Source\n\
         - Original: `{input_path}`\n\
         - Generated: `main.py`\n\n\
         ## Requirements\n\
         - Python >= 3.11\n\
         - [uv](https://github.com/astral-sh/uv)\n\n\
         ## Run\n\
         ```\n\
         uv run main.py\n\
         ```\n",
        name = name,
        input_path = input_path
    )
}

## Phase D — `--check` オプション

`cmd_transpile` の末尾（ファイル書き込み後）に追加:

```rust
if do_check {
    let check_target = if out_dir.is_some() {
        format!("{}/main.py", out_dir.as_ref().unwrap())
    } else {
        out.clone()  // --out または デフォルト出力先
    };
    let status = std::process::Command::new("python")
        .args(["-m", "py_compile", &check_target])
        .status();
    match status {
        Ok(s) if s.success() => {
            println!("fav transpile: syntax ok ({})", check_target);
        }
        Ok(_) => {
            eprintln!("error: python syntax check failed: {}", check_target);
            std::process::exit(1);
        }
        Err(_) => {
            // python が見つからない場合はスキップ（Windows 等で `python` がない場合）
            eprintln!("warning: python not found, skipping --check");
        }
    }
}
```

## Phase E — `--run` オプション

`--check` の後に追加:

```rust
if do_run {
    let run_dir = out_dir.as_deref().unwrap_or(".");
    let status = std::process::Command::new("uv")
        .args(["run", "main.py"])
        .current_dir(run_dir)
        .status();
    match status {
        Ok(s) => {
            if !s.success() {
                std::process::exit(s.code().unwrap_or(1));
            }
        }
        Err(_) => {
            eprintln!("error: uv not found (install: pip install uv)");
            std::process::exit(1);
        }
    }
}
```

## Phase F — テスト（driver.rs 末尾 v11700_tests）

テストはファイル I/O を伴うため `tempfile::TempDir` を使用する。

```rust
#[cfg(test)]
mod v11700_tests {
    use tempfile::TempDir;
    use std::fs;

    fn write_fav(dir: &TempDir, name: &str, src: &str) -> String {
        let path = dir.path().join(name);
        fs::write(&path, src).unwrap();
        path.to_str().unwrap().to_string()
    }
}
```
