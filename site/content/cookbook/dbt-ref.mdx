# dbt モデル参照

dbt で管理されたモデルを Favnir パイプラインから参照し、結果を取得します。

```favnir
import runes/dbt

stage ReadDbtModel: String -> List<Row> !Db = |model_name| {
  bind result <- DBT.run_and_fetch[Row](config.dbt_project, model_name)
  Result.ok(result)
}
```
