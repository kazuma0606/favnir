# Rune カタログ

`fav search` で検索できる公式 Rune パッケージの一覧です（v24.5.0 時点、50 パッケージ）。

```bash
fav search                  # 全パッケージを一覧表示
fav search "bigquery"       # 名前または説明で検索
fav install <name>          # インストール
```
